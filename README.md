
This is a basic Student Record management sysytem.

A modern, web-based Student Management System built with Python, Flask, and Object-Oriented Programming (OOP). This application allows you to manage student records (Add, View, and Delete) with persistent storage using a JSON** database.

 Features
OOP Architecture: Uses Python classes to represent students and management logic.
Web UI: Clean and responsive interface built with HTML/CSS.
Persistent Storage: Data is saved in a `students.json` file so no data is lost on restart.
RESTful API: Uses Flask routes to handle data communication.



 Tech Stack
 Backend: Python 3.x, Flask
Frontend:HTML5, CSS3, JavaScript (Fetch API)
Database: JSON (File-based)
